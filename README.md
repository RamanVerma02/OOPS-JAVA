# ▶️ How to Run the Programs

### 1️⃣ Clone the repository

```bash
git clone https://github.com/RamanVerma02/OOPS-JAVA.git
```

### 2️⃣ Navigate to project folder

```bash
cd OOPS-JAVA
```

### 3️⃣ Compile Java program

```bash
javac ProgramName.java
```

### 4️⃣ Run the program

```bash
java ProgramName
```
